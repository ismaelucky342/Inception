✓ Contenedor ejecutándose correctamente

Este HTML está siendo servido por python3 -m http.server desde dentro del contenedor.

Conceptos de este nivel

  • EXPOSE 8000 — documenta el puerto
  • -p 8080:8000 — mapea host:contenedor
  • ENTRYPOINT + CMD — proceso en foreground
  • PID 1 = el proceso principal del contenedor
