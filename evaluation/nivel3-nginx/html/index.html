🐳 Ejemplo 3 — Nginx con conf custom

Nginx sirviendo con configuración propia

Puerto del contenedor: 80 → mapeado a 8080 en el host

Lo importante

  • daemon off — mantiene nginx en foreground (PID 1)
  • nginx.conf propio — reemplaza la config default
  • Sin esto el contenedor moriría inmediatamente

este patrón (proceso en foreground) es la base de todos los servicios en Inception 42
